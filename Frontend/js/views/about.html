About Stock Manager

Project name: Stock Manager
Project Overview
Stock Manager is a Java-based application for effectively managing stocks and customer information. The system provides functionality to update, add, and remove stocks; to manage customer information and track their interactions with the stocks; and to handle stock purchases and sales.
Project Features
  • Stock Management: Add a new stock, remove by symbol, or update stock details (price/quantity).
  • Customer Management: Register new customers and remove existing ones by ID.
  • Stock Transactions: Customers can buy or sell stocks. The application updates relevant stock quantities and records transaction history per customer.
  • Persistence: Data is stored in persistent Oracle database tables, loaded and written as needed.
Technical Details
  • Backend: Java (Spring Boot), RESTful APIs, Oracle database
  • Frontend: Oracle JET (Knockout.js), HTML5, CSS3
  • Integration: JDBC for database access, KO for dynamic UI binding
